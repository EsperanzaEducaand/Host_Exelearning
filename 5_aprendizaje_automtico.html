5.- Aprendizaje Automático

Vamos a crear un sistema de Inteligencia Artificial capaz de identificar a los diferentes miembros del grupo e indicarnos con que porcentaje de seguridad nos ha identificado. El trabajo se divide en 2 fases:

FASE 1 - Grupal
En esta fase crearemos un modelo de datos a partir de las fotos de todos los miembros del grupo y lo probaremos. Nos debemos descargar un archivo JSON con el modelo de datos.

FASE 2 - Individual
En esta segunda fase cargaremos el archivo JSON, volveremos a crear el modelo de datos y en Scratch crearemos nuestra aplicación que haciendo uso de la webcam identificara a cada uno de los miembros de nuestro grupo.

AYUDA
Enlace a la web LearningML: https://web.learningml.org/

Videotutorial de como usar la web LearningML.


© learningml - Youtube

ENTREGA
Cada miembro de debe entregar de manera individual:

  • Archivo JSON con el modelo de datos generado en LearningML.
  • Archivo SB3 del proyecto de Scratch.
